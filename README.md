# model-viewer

GUI app to view glTF models and more.

## Compiling

### Prerequisites

- Node.js v18.16.0 LTS
- Rust v1.69.0
- pnpm v8.4.0

Install anything else required [here](https://tauri.app/v1/guides/getting-started/prerequisites).

```
pnpm install
```

Build for production

```
pnpm tauri build
```

Launch in dev mode

```
pnpm tauri dev
```
